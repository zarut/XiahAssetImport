
# XiahAssetImport

This project is a Unity-based tool for viewing in-game character and object models from the game Xiah.
## Requirements

For compile, you need

    Visual Studio (2012 or later)
    Unity 2018.4.15f1

## Known problems
It wont load textures automatically, you need to use XPK Unpacker to find correct texture and add it yourself. Also doesnt support skeleton data so no animation viewing yet.
